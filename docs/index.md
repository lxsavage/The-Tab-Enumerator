Number your tabs for quick keyboard navigation.

Do you regularly use ⌘+number/CTRL+number keyboard shortcuts in tab navigation,
but have lots of tabs so you can't easily keep track of which is which? This
extension adds numbering to each tab when you hold down the modifier key (CTRL
or ⌘) so that you can quickly jump to a tab by number.
